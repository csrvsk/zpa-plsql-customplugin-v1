Usar uma condição LIKE sem um curinga (% ou _) é suspeito. Um desenvolvedor pode supor que o curinha foi esquecido ou se deveria ser um teste de igualdade.

Código em desconformidade

if (last_name like 'Smith') ...

Código correto

if (last_name like 'Smith%') ...

-- or
if (last_name = 'Smith') ...
